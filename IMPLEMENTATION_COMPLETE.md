## 🚀 How to Use

### Running the Project
```bash
npm install
npm run dev
# or
pnpm install
pnpm dev
```

Visit `http://localhost:3000`

### Customizing Content
Each component has clear, editable content sections:
- Headlines and copy in JSX
- Images can be replaced in `public/images/`
- Colors easily adjustable in `globals.css`
- Animations can be tweaked via keyframes

### Deploying
```bash
npm run build
npm run start
# or deploy to Vercel/your platform
```
